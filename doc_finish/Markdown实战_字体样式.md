在写作时，有时我们需要提醒读者某些内容，此时醒目的字体样式可以提供帮助。

|Markdown写法|HTML|渲染效果|
|:--|:--|:--|
|\*斜体\* 或 \_斜体\_|\<em>斜体\</em>|_斜体_|
|\*\*粗体\*\* 或 \_\_粗体\_\_|\<strong>粗体\</strong>|__粗体__|
|\*\*\*粗斜体\*\*\* 或 \_\_\_粗斜体\_\_\_|\<strong>\<em>斜体\</em>\</strong>|___粗斜体___|
|\~\~删除线\~\~|\<del>删除线\</del>|~~删除线~~|
|无|\<u>下划线\</u>|<u>下划线</u> |
